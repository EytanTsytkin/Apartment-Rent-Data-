# Apartment-Rent-Data-
A script that gathers information about apartments for rent in Israel.
      
Current Sources: 
      Yad 2
 
Metrics: 
      Rent
      Adress, including floor 
      Size (in squared meters)
      Number of bedrooms
 
 Future goals:
      - Creating a Database
      - Automate a daily update
      - Map display
      - Expand to further sources
